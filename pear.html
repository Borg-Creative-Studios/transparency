    Project Pear

    Project Pear is a development project to produce an easy to setup user customizable experience that respects privacy.

    Many user customizable experiences utilize many cookies and tracking systems to record and deliver the user's settings. Project Pear is capable of doing both without tracking the user. 

    Project Pear utilizes DOM, JavaScript, and local browser storage to save and load a customized user experience.

    Project Pear is built on BCS Author, an open source browser based writing app.
